# automatoooor
Automate actions in crypto.  From keeping an eye on informaiton to performing sells, stakes, pair breaks, auto claims, autocompound, etc...

Networks supported: Harmony, Fantom, Avalanche, Binance Smart Chain, Ethereum, Aurora, Polygon, RSK and Telos

###Instructions
1. Clone the repository
2. Run "npm install"
3. Create an .env file and add your information with this format
```PUBLIC_KEY_HARMONY=
PRIVATE_KEY_HARMONY=

PUBLIC_KEY_FANTOM=
PRIVATE_KEY_FANTOM=

PUBLIC_KEY_AVAX=
PRIVATE_KEY_AVAX=

PUBLIC_KEY_BSC=
PRIVATE_KEY_BSC=

PUBLIC_KEY_ETH=
PRIVATE_KEY_ETH=

PUBLIC_KEY_AURORA=
PRIVATE_KEY_AURORA=

PUBLIC_KEY_POLY=
PRIVATE_KEY_POLY=

PUBLIC_KEY_RSK=
PRIVATE_KEY_RSK=

PUBLIC_KEY_TELOS=
PRIVATE_KEY_TELOS=
```
4. Run "npm start"
5. Observe as it starts automating your crypto
